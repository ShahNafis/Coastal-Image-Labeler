A labeling tool for Coastal Images

Docs

Preprocessing

Labeled images are important for supervised machine learning research. There are many well known labeled image databases, but these existing databases tend to focus on general features . Our goal with this project is to develop a discipline-specific database of labeled images that is relevant for coastal scientists.

Tag Collaboratively

We wanted a tool to easily accommodate multiple users labeling a single images to ensure correct labeling via consensus.

Easy to use

A server that provides users with images to be labeled using a given set of questions. We designed the labeler to be hosted on a virtual machine, exposed via a web address. Users interactively label images, with labels written to a database and exported later by an admin.
